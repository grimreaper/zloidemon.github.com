Ставим порт:

```bash
%sudo make -C /usr/ports/sysutils/fusefs-ntfs install clean
%sudo echo 'fusefs_enable=\"YES\"'>>/etc/rc.conf
```

Удоляем и меняем:

```bash
%sudo rm /sbin/mount_ntfs
%sudo cp /usr/local/share/hal/mount-fuse /sbin/mount_ntfs
```
